##How to Run the Program
1. Make sure Python is installed.
2. Install PySide6 by typing:
   pip install PySide6
3. Place these files in the same folder:
   - converter.py
   - house.png
4. Run the program:
   python converter.py

##Program Description
This program is a measurement converter built using PySide6.

The user enters a number and selects:
- Inches to Meters
- Meters to Inches

Then clicks Convert to see the result.

##Features
- Checks if input is empty
- Checks if input is numeric
- Requires positive values
- Displays result rounded to 3 decimal places
- Clear button resets everything
- Exit button closes the program

##Test Cases
